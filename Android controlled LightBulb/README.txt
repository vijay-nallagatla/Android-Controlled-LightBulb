LED Controller Using Windows Form Application and Arduino Circuit Board 

Software Requirement

1) Visual Studio 2010 onwards
2) Arduino Sketch

Technical Requirements

1) C# windows Application (SerialPort Communication)
2) Arduino Program (To communicate with PC)
3) Arduino Board and LED Circuit


Working

1) Standalone Windows Application is used to Send Commands to Arduino Circuit Board
2) Arduino Receives the command from windows Application over COM PORt and ACT Accordingly
3) PC APPLication Sends Command to turn LED ON or OFF
